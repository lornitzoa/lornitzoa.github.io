Projects

BeHaver

BeHaver is a positive reinforcement app for parents and their children. This was my final project for my WDI program through General Assembly. I drew from my background in behaviorism to create a data driven application using React, Ruby on Rails and PostgresSQL.

Nutrition Tracker

This was the first Javascript project assigned during my GA WDI program. With this opportunity I built a nutrition tracker using jQuery, HTML, CSS and a third party API through Nutrix.com. I pushed myself to learn how to do drag and drops as well as use local storage to save user data.

Tic Tac Toe

This was a weekend assignment from my GA WDI program that required me to build a tic tac toe game using jQuery. I was particularly excited to figure out how to display lines to show where a player won. Figuring out the logic of testing for win conditions was tricky and I pushed myself to test for diagonal wins.
