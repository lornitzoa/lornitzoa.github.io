BeHaver

BeHaver is a positive reinforcement app hosted on Heroku for parents and their children. This was my final project for my WDI program. I drew from my background in behaviorism to create a data driven application using React, Ruby on Rails and
PostgresSQL.

Planit



Planit is an event and goal planning application. This collaborative project was built with a MEAN stack, used a third party API through thenounproject.com, and is hosted on Heroku. We used Trello to assist with project management and GitHub for code collaboration.

The Pantry



The Pantry is a MEAN stack app hosted on Heroku, designed as a nutrition data interface. Rather than tracking nutrition intake like the Nutrition App, this is designed primarily as a way to collect and store nutrition data, and was specifically geared toward collecting
FODMAP data.
